# 🍜 Craving & Creation

Craving & Creation is a recipe website built using HTML, CSS, and JavaScript. Users can explore recipes, search dishes, save favorites, add custom recipes, and enjoy an interactive flip-card experience.

## ✨ Features

* Recipe Categories
* Search Functionality
* Favorites System
* Add Your Own Recipes
* Food Reviews
* Responsive UI Design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
